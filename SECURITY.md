# Security

## Principles
- Never commit secrets or production `.env` values.
- Treat legal documents and client messages as confidential.
- Restrict API CORS and authentication in production.
- Keep AI outputs labeled as support tools, not legal advice replacements.

## Recommended Headers
- `Strict-Transport-Security`
- `Content-Security-Policy`
- `X-Content-Type-Options`
- `Referrer-Policy`
- `Permissions-Policy`
