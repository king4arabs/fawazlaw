# SKILLS System

The SKILLS folder is the operating playbook for Fawazlaw.sa. Each file defines how a cross-functional discipline should make decisions, review work, and validate delivery in this repository.

## How to use
1. Start every task by reading the relevant SKILLS file.
2. Apply the checklist before implementation and during review.
3. Update docs, status, roadmap, changelog, decisions, and tests when work changes repository behavior.
4. Preserve existing work unless it is unsafe, obsolete, or clearly duplicated.
5. Keep Saudi-first, Arabic/English, RTL/LTR, confidentiality, security, and production readiness in scope.

## Definition of done for any skill
- The work is aligned with business/legal context.
- The implementation is secure, tested where possible, documented, and reversible.
- Validation results and known risks are recorded.
