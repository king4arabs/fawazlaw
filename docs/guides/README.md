# Guides

This folder holds how-to guides for common contributor tasks. Add files as needed; suggestions:

- `add-a-service-page.md` — how to add a new legal service landing page.
- `add-an-article.md` — how to publish a practitioner-reviewed article.
- `add-a-translation.md` — how to add an English (or other) translation.
- `add-an-api-endpoint.md` — how to add and document a new API endpoint.
- `update-dependencies.md` — how to safely bump a dependency.
- `triage-an-incident.md` — how to handle a new incident end-to-end.
