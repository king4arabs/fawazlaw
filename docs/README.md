# Fawazlaw.sa Deep Documentation

This `docs/` tree holds deep guides that complement the root operating-system documents (README, ARCHITECTURE, SECURITY, etc.) and the discipline playbooks in `SKILLS/`.

## Map
- `wiki/` — top-level wiki: product/technical/operations/release/security/Saudi-GCC overviews.
- `api/` — API reference and planning (endpoints, auth, errors, webhooks, examples).
- `sdk/` — SDK planning and language-specific notes (JS/Python/PHP).
- `guides/` — how-to guides for common contributor tasks.
- `runbooks/` — operational runbooks (release, incident, backup).
- `releases/` — per-release notes.
- `compliance/` — PDPL ROPA, privacy, AI provider notes.
- `design-system/` — tokens, components, RTL/LTR patterns.
- `product/` — product specs archive.
- `business/` — pricing, packaging, partnerships.
- `growth/` — marketing experiments, content calendar.
- `security/` — secrets, threat model, hardening guides.
- `testing/` — test strategy detail, fixtures, conventions.
- `deployment/` — environment-specific deployment notes.

## Conventions
- Markdown only.
- Arabic content lives alongside English with a clear filename suffix when relevant (`*.ar.md`).
- Update the relevant root document when a guide changes a standard.
