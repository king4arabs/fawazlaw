# Saudi / GCC Readiness (Wiki)

See `SKILLS/saudi_gcc_readiness.md` for the working playbook. Highlights:

- Arabic-first UI; RTL default.
- Hijri/Gregorian display where the audience expects both.
- SAR currency by default; VAT shown clearly.
- Saudi mobile format validation; mada / Apple Pay / STC Pay via MyFatoorah.
- Saudi-region or GCC-region hosting preference for production personal data.
- Practitioner-reviewed Saudi legal terminology.

## GCC Expansion
- New countries ship as separate surfaces (`/sa`, `/ae`, …) — never a single page stretched across jurisdictions.
- Bilingual parity (AR/EN) is a baseline expectation for enterprise/expat audiences.
